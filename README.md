vim 配置

vim plugin 
____
1. git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim

golang config
____
1. 添加 Plugin 'fatih/vim-go'
2. 执行插件安装 `:PluginInstall`
3. 安装binaries `:GoInstallBinaries`
